## Usage

```mdx
import MasonryGallery from '@/components/MasonryGallery.astro'

<MasonryGallery
  columns={3}
  images={[
    { src: "/posts/my-post/photo1.jpg", alt: "...", caption: "optional" },
    { src: "https://external.com/img.jpg", alt: "..." },
  ]}
/>
```

`src` accepts any URL or `/public` path. `columns` defaults to `3`. Click any image to open the lightbox.
